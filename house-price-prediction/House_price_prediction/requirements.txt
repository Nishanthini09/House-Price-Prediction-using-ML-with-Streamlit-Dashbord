#Required Libraries
Here are the required Python libraries along with their versions:

    numpy: For numerical operations
    pickle: For loading the saved machine learning model
    streamlit: For building the web application interface
    streamlit-option-menu: For creating an option menu in the sidebar
    Pillow: For handling image loading
